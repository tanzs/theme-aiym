# AIYM Halo 响应式主题

一个现代化的响应式 Halo 博客主题，支持浅色/深色模式、资源库、搜索和完整移动端适配。

## 功能特性

- 响应式设计，完美适配桌面端和移动端
- 浅色/深色模式切换，支持跟随系统
- 博客文章展示，支持热门文章、分类筛选
- 资源库管理，支持下载、版本、平台等扩展字段
- 独立搜索页面，支持关键词搜索
- 文章目录、代码复制、图片预览
- 评论功能（Halo 评论扩展点）
- SEO 优化，支持 Open Graph
- 无障碍访问支持

## 安装

1. 下载主题包
2. 在 Halo 后台 → 外观 → 主题 → 安装
3. 选择主题包文件上传
4. 安装完成后启用主题

## 配置

### 基础设置

- **网站 Logo**: 上传 Logo 图片
- **默认封面**: 文章未设置封面且随机封面图库为空时使用
- **主导航菜单**: 选择 Halo 主菜单
- **搜索页面**: 选择搜索独立页面
- **工具页面**: 选择在线工具入口独立页面
- **关于页面**: 选择关于独立页面

### 内容设置

- **博客根分类**: 首页只显示此分类下的文章
- **资源根分类**: 资源库只显示此分类下的资源
- **每页文章数**: 列表页每页显示的文章数量
- **热门文章数**: 首页显示的热门文章数量
- **推荐资源数**: 资源库显示的推荐资源数量

### 外观设置

- **默认模式**: 跟随系统、浅色或深色
- **品牌色**: 主题的主要品牌色，默认 `#42A5F5`
- **显示主题切换按钮**: 是否在导航栏显示明暗切换按钮
- **圆角级别**: 无圆角、小圆角、中等圆角、大圆角
- **启用动画**: 是否启用页面加载和交互动画
- **随机封面图库**: 文章无封面时从这些图片中随机选取

### 资源设置

- **下载按钮文字**: 资源下载按钮显示的文字
- **无下载提示**: 资源没有下载地址时的提示文字
- **显示免责声明**: 是否在资源详情页显示免责声明
- **免责声明内容**: 资源详情页底部显示的免责声明

### SEO 与备案

- **站点描述**: 留空时使用 Halo 系统设置中的站点描述
- **站点关键词**: 用于全站默认 keywords
- **OG 默认图片**: 文章无封面时用于社交分享卡片
- **ICP 备案号 / ICP 备案链接**: 配置后显示在页脚并跳转到对应备案页
- **公安备案号 / 公安备案链接**: 配置后显示在页脚并跳转到对应备案页

## 内容约定

### 分类

- 每篇博客文章必须关联博客根分类
- 每篇资源文章必须关联资源根分类
- 可以关联一个或多个子分类

### 资源字段

资源扩展字段使用文章注解：

- `theme.aiym.fun/download-url`: 下载地址
- `theme.aiym.fun/icon-url`: 图标地址
- `theme.aiym.fun/version`: 版本号
- `theme.aiym.fun/platform`: 平台（逗号分隔）

为兼容现有内容，读取时同时接受：

- `downloadUrl`
- `iconUrl`
- `version`
- `platform`

## 开发

### 环境要求

- Node.js >= 18
- npm 或 yarn

### 安装依赖

```bash
cd theme-aiym
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 类型检查

```bash
npm run type-check
```

### 代码检查

```bash
npm run lint
```
